[*]-[MyScanner] Adalah alat untuk mengetahui kerentanan sebuah situs atau website, sehingga anda dapat melaporkan hal itu ke developer website tersebut
[*]- Jadi tujuan pembuatan alat ini adalah untuk melakukan scanner terhadap website, agar bisa melaporkan kerentanan website kepada developer atau admin. Saya tidak bertanggung jawab atas penyalah guanaan alat ini
[*]- Installasi dan penggunaan:
my-scanner
cd modules
chmod +x *
./install.sh 
[*]- Untuk menjalankannya:
sh install.sh -url
[*]- Tinggal ikuti langkah selanjutnya ok.....!!!
